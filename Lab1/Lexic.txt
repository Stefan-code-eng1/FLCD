Alphabet:

a. Upper (A-Z) and lower case letters (a-z) of the English alphabet

b. Underline character '_';

c. Decimal digits (0-9);

Lexic:

      a.Special symbols, representing:

	- operators: 

	- separators: ( ) { } ; , 

	- reserved words: adapt basic f type give create read give write INT STRING BOOL as times less plus 
	minus div eq TRUE FALSE of for
    

      b.identifiers

  	-a sequence of letters and  digits, such that the first character is a letter; the rule is:

    		identifier = letter | letter{letter}{digit}

    		letter = [A-Z] | [a-z]

    		digit = [0-9]


      c.constants

	1.integer - rule:

      		noconst = "plus"no|"minus"no|no

      		no = digit{no}

	2.character

    		character = 'letter'|'digit'

	3.string

      		constchar = "string"

      		string = char{string}

      		char = letter|digit
